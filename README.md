## Run Locally
Clone the project
```bash
  https://github.com/Asanda001019/BOOLEAN-WORK.git
```
Go to the project directory
```bash
  cd BOOLEAN-WORK
```
Install dependencies
```bash
  npm install
```
Start the react server
```bash
  npm run dev
```
## Tech Stack
**Client:** React JS, HTML5, JavaScript, CSS3
